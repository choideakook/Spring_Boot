# 데이터베이스 CRUD

CRUD 는 대부분의 소프트웨어가 가지는 기본적인 데이터 처리기능인

Create (생성), Read (읽기), Update (갱신), Delete (삭제) 의 줄임말이다.

DB 를 사용하기위한 기초적인 Query 형식과

사용자 인터페이스가 갖추어야 할 기능을 가리키는 용어로서도 사용된다.

## ✏️ SQL 문과의 대응

| 이름 | 조작 | SQL |
| --- | --- | --- |
| Create | 생성 | INSERT |
| Read, Retrieve | 읽기, 인출 | SELECT |
| Update | 갱신 | UPDATE |
| Delete | 삭제 | DELETE |
