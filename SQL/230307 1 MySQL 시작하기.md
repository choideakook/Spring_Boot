# MySQL 시작하기

## ✏️ 기본 세팅과 접속

[🔗 기본 세팅](https://velog.io/@taeate/Sequel-Pro-MySQL-DB-관리하는법)

- ***서버 시작***

```
mysql.server start
```

- ***로그인***
    - 명령후 pw 입력
    - 명령후 mysql> 이 나오면 접속 완료

```
mysql -u root -p
```

### 📍 **sequel pro**

1. **sequel pro 를 실행후 항목을 입력해준다.**
    - Name : 프로젝트명
    - Host : data base 주소
    - User name : 접속할 my sql id
    - Password : my sql pw
2. Connect 전에 Test Connection 으로 연결 테스트를 해준다.
3. Connect 로 연결시켜준다.
